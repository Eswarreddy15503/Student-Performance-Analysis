# Student Performance Analysis

## Project Overview

This project analyzes student academic performance using demographic and assessment data from 1000 students. The objective is to identify factors influencing student success and detect students who may be academically at risk.

## Objectives

* Perform data exploration and cleaning
* Analyze factors affecting performance
* Create six visualizations
* Conduct correlation analysis
* Identify at-risk students
* Provide recommendations for school administrators

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Key Findings

* Higher parental education is associated with better student performance.
* Test preparation significantly improves scores.
* Reading and writing scores show strong correlation.
* Female students perform better in reading and writing.
* 18.8% of students were identified as academically at risk.
